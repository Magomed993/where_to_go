# Куда пойти

Проект предназначенный для поиска мест, куда можно сходить по городу Москва.

![Снимок экрана от 2024-12-19 00-38-30](https://github.com/user-attachments/assets/ae2eb467-b83c-41ed-85df-71dd8e70555e)
## Демо-версия сайта

[magom.pythonanywhere.com](https://magom.pythonanywhere.com/)

## Как запустить

Для запуска сайта вам понадобится Python версии 3.10.

Скачайте код с GitHub. Установите зависимости:

```
pip install -r requirements.txt
```

Создайте базу данных SQLite

```
python3 manage.py migrate
```

## Переменные окружения

Часть настроек проекта берётся из переменных окружения. Чтобы их определить, создайте файл `.env` рядом с `manage.py` и запишите туда данные в таком формате: `ПЕРЕМЕННАЯ=значение`.

Доступны 3 переменные:
- `DEBUG` — дебаг-режим. Поставьте `True`, чтобы увидеть отладочную информацию в случае ошибки.
- `SECRET_KEY` — секретный ключ проекта. Данный ключ автоматически генерируется при создании проекта
- `ALLOWED_HOSTS` — см [документацию Django](https://docs.djangoproject.com/en/3.1/ref/settings/#allowed-hosts)
